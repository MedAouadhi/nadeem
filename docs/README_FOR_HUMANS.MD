# Documentation for Humans

This directory contains the documentation and intelligence for this project.

## Purpose

The `docs/` folder is the central repository for project knowledge. It contains everything from technical diagrams to high level workflows.

## Navigation

- Use [INDEX.md](./INDEX.md) to explore the subdirectories.
- Subdirectories are organized by theme (e.g., `workflows/` for processes, `diagrams/` for visual architecture).

## Project Intelligence

The `project-intelligence/` directory stores captured context and learnings that help maintain project continuity.
